# New Home Search Docs

## Purpose
This repo is the source of truth for my apartment search in Hollywood / West Hollywood.

## Working model
- GitHub is the canonical project space.
- Structured apartment data lives in `data/`.
- Human-readable docs live in `docs/`.
- Notion is a phone-friendly mirror for viewing key apartment/unit data.
- ChatGPT is used for breadth-first research, extraction, and organization.
- Claude is used for repo edits, deeper critique, and final refinement.

## Search priorities
- Preserve or exceed Camden Hollywood level amenities
- Stay in or near Hollywood / West Hollywood
- Prioritize quieter, more stable, more adult community/building vibe
- Strong, trustworthy, no-nonsense leasing and maintenance team
- Prefer newer construction or very well-kept interiors
- Cat friendly
- Target rent ideally within $3,800 to $4,700, stretch to $5,300 all-in only if strongly justified

## Canonical structured files
- `data/buildings.csv`
- `data/tours.csv`
- `data/contacts.csv`
- `data/decisions.csv`
